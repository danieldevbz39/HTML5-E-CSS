Seletores Básicos

Estes são os "pães com manteiga" do CSS, usados em 90% do tempo.

* : Seletor Universal. Seleciona absolutamente todos os elementos da página. Muito usado para "resetar" margens e paddings.

h1 : Seletor de Tipo (Tag). Seleciona todos os elementos que possuem essa tag HTML específica (ex: todos os parágrafos, todos os títulos).

.minha-classe : Seletor de Classe. Seleciona todos os elementos que possuem o atributo class="minha-classe". É o mais versátil.

#meu-id : Seletor de ID. Seleciona o único elemento com o atributo id="meu-id". IDs devem ser únicos por página.

Seletores Combinadores

Eles definem a relação de hierarquia entre os elementos.

div p : Descendente. Seleciona todos os

que estejam dentro de uma

, não importa quão fundo na hierarquia.

div > p : Filho Direto. Seleciona apenas os

que são filhos imediatos da

.

h1 + p : Irmão Adjacente. Seleciona o primeiro

que aparece imediatamente após um h1.

h1 ~ p : Irmão Geral. Seleciona todos os

que sejam irmãos de um h1 (que estejam no mesmo nível hierárquico e venham depois dele).

Seletores de Atributo

Úteis quando você quer estilizar elementos com base em propriedades específicas.

input[type="text"] : Seleciona apenas campos de entrada que sejam do tipo texto.

a[href^="https"] : Seleciona links cujo endereço comece com "https" (links seguros).

img[alt] : Seleciona apenas imagens que possuam o atributo alt definido.

Pseudo-classes e Pseudo-elementos Eles estilizam estados especiais ou partes específicas de um elemento.

a:hover : Seleciona o link apenas quando o usuário passa o mouse sobre ele.

li:nth-child(even) : Seleciona todos os itens de uma lista que ocupam posições pares (2, 4, 6...).

input:focus : Seleciona o campo que está com o cursor ativo no momento.

p::first-line : Seleciona e estiliza apenas a primeira linha de um parágrafo.
